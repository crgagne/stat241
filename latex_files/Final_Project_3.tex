    \subsubsection{Multi-Subject Potts
Model}\label{multi-subject-potts-model}

The states of the system z, are assumed be distributed according to a
Gibbs distribution.

\[ p(Z) = 1/P exp(-U(Z)) \] \[ p(Z) = 1/P exp(-U(Z)) \]

There are nodes in group and nodes in each subjects map.

How to sample from the Potts model.

\[ p(y_s|y_{-s},x) = p(y)/p(x|y_s)p(y_{-s}) \]

Initialize Random States for All Nodes in both group and subject 1.
sample from group using Gibbs Sampling -
\[ p(y_s|y_{-s},x) = 1/P exp(-U(Z))\] - where U(Z) in this case is
\[ U(Z) = \alpha \sum_j \delta(y_j,y_j')  + \beta \sum_n \delta(y_s,y_n)  +\]
2. sample from each subject using Gibbs Sampling -
\[ U(Z) = \alpha \delta(y_j,y_j')  + \beta \sum_n \delta(y_s,y_n)  -kux  - logCp \]
- Only use the liklihood components
